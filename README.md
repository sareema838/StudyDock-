# StudyDock

**StudyDock** is a simple, distraction-free web study planner designed for students who struggle with managing daily tasks and staying consistent. It helps students plan, track, and complete study tasks efficiently with a clean, intuitive interface.

## Features

- Add study tasks with a title
- Mark tasks as complete or pending
- Delete tasks
- Daily progress tracker (completed vs total tasks)
- Data persists locally in the browser
